Conversor Base64 Avançado com Criptografia Opcional

📤 Arquivo → Base64: Preserva nome, extensão, tamanho, tipo MIME, hash SHA-256 e conteúdo do arquivo, tudo encodado em Base64. Opcionalmente, pode criptografar o resultado com senha usando AES-GCM.

📥 Base64 → Arquivo: Valida o formato Base64, decodifica o JSON, verifica a integridade com hash SHA-256 e permite download do arquivo original. Se os dados estiverem criptografados, descriptografa automaticamente.

🔐 Criptografia Opcional: Use criptografia AES-GCM com PBKDF2 para proteger seus dados Base64. A criptografia é totalmente opcional e pode ser habilitada/desabilitada conforme necessário.

🔐 Validação Robusta: Verifica formato Base64, estrutura JSON e integridade antes de processar.

🛡️ Segurança: Todo processamento é local no seu computador, nenhum dado é enviado para servidores.
