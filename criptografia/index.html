Criptografia Segura AES-GCM

🔐 Algoritmo: AES-GCM com chave de 256 bits, considerada uma das formas mais seguras de criptografia.

🛡️ Proteção por Senha: Usa PBKDF2 com 100.000 iterações para proteger contra ataques de força bruta.

🔑 Chave Única: Cada criptografia gera uma nova chave aleatória, garantindo máxima segurança.

⚠️ Importante: Guarde sua senha em local seguro. Sem ela, é impossível recuperar os dados criptografados.

🔒 Privacidade: Todo processamento é feito localmente no seu navegador, nenhum dado é enviado para servidores.
